Phython
==========
> 파이썬에 대해 공부한 내용들

# 자료형

- 숫자 자료형

- 문자열 자료형
    - " " 또는 ' ' 안에 작성
- Boolean 자료형
    - 조건에 따라 True, False 를 나타냄
***
# 변수
값을 저장할 때 사용하는 식별자
***
# 연산자
+ `+` 덧셈
+ `-` 뺄셈
+ `*` 곱셈
+ `/` 나눗셈
+ `%` 나머지
+ `>, >=, <, <=` 크기비교 
+ `==` 등호
+ `!=` 부정
***
# 함수
+ 숫자처리함수
    +  `abs()` 절대값
    + `pow(a , b)` a의 b승 
    + `max(a, b)` a와 b를 비교하여 최댓값을 반환
    + `min(a, b)` a와 b를 비교하여 최솟값을 반환
    + `round(a)` 소수 a를 반올림하여 반환
        +  math 라이브러리
            + `floor(a)` a의 값을 내림하여 반환
            + `ceil(a)`a의 값을 올림하여 반환
            + `sqrt(a)` a의 제곱근을 반환
+ 랜덤함수
    + `random()` 0.0 ~ 1.0 미만의 임의의 값 생성
    + `randrange(a, b)` a ~ b 미만의 임의의 값 생성
    + `randint(a, b)` a ~ b 이하의 임의의 값 생성
    + `lower` 모든 문자가 다 소문자로 출력
+ 슬라이싱
    + 문자열 자료형`[a:b]` a부터 b까지 출력
    + 문자열 자료형 `[:b]` 처음부터 b까지 출력
    +문자열 자료형 `[a:]` a부터 끝까지 출력
+ 문자열처리함수
    + `upper` 모든 문자가 다 대문자로 출력
    + `isupper` 문자열의 0번째 위치가 대문자인지 판별
    + `len` 문자열의 길이를 반환
    + `replace ("a", "b")` a를 b로 변환
***
# 문자열
+ 문자열포맷
    + int형일땐 `%d`
    + String형일땐 `%s`
    + char형일땐 `%c`
+ 탈출문자
    + `\n` : 줄바꿈
    + `\t` : tab
    + `\b` : Backspace
    + `\" \` : 문장 내에서 따옴표
    + `\\` : 문장 내에서 \
    + `\r` : 커서를 맨 앞으로 이동
***
# 분기문
+ if문 사용법 
    + `if 조건:
    실행 명령문`
+ 또 다른 조건 사용법
    + `elif 조건:
      실행 명령문`
+ 또또 다른 조건 사용법
    + `else:
    실행 명령문`
+ 또다른 사용법
    + 조건 사이에 `and,or` 등을 사용할 수 있음
***
# 반복문
+ for문
    + `for name in range (a)` 0부터 5까지 반복


+ while문
    + `while 조건`
    조건이 만족될때까지 반복
